MySql常用语法

发表于 2019-10-18
本文字数： 1.2k 阅读时长 ≈ 1 分钟

#—–MySql数据库操作基础—–
###显示数据库
show databases;

判断数据库是否已存在数据库有的话先删除

drop database if exists 库名;

###创建数据库
create database 库名;

删除数据库

drop database 库名;

使用该数据库

use 库名;

显示数据库中的表

show tables;

先判断表是否存在,存在先删除

drop table if exists 表名;

创建表

create table 表名(
字段名1 字段类型1（规定长度）,
字段名2 字段类型2（规定长度）,
...
)default charset=utf8;

删除表

drop table 表名;

查看表的结构

describe 表名;  #可以简写为desc 表名;

插入数据

insert into 表名(要插入的字段名) values(插入的字段值);
insert into 表名 values(插入表中所有字段的值);

查询表中的数据

select * from 表名;
select id,name from 表名;

修改数据

update 表名 set 字段名 = 新的字段值

修改指定数据

update 表名 set 字段名 = 新的字段值... where 条件表达;

删除数据

delete from 表名 where 条件;

约束

在类型后面添加字段约束
唯一约束：unique
非空约束：not null
主键约束：primary key
唯一+非空=主键约束
一般数字类型自增长策略：auto_increment

外键约束不能在字段后直接加，要在所有字段后用逗号隔开
外键约束：foreign key（要添加约束的字段） references 关联的表名(关联的字段名);  外键必须是另一张关联表的主键

查询语句格式

select [字段列表，表达式，函数，*] from 表名; *表示全部数据
 distinct 去重

and 且 条件1,2都符合

select * from 表名 条件1 and 条件2;

or 或 符合条件1或条件2的数据

select * from 表名 where 条件1 or 条件2;

between…and..在什么什么之间的数据

select * from 表名 where 字段名 between 字段1 and 字段2;

in 查询在什么什么选项中的数据查询出来

not in 是in的取反把不等于的取出来

select * from 表名 where 字段名 in (字段1,字段3,字段5);

order by排序

#把StudentInfo表按年纪从小到大排序
SELECT * FROM StudentInfo ORDER BY age ASC;

# 把StudentInfo表按年纪从大到小排序
SELECT * FROM StudentInfo ORDER BY age DESC;

GROUP BY 语法

GROUP BY 语句根据一个或多个列对结果集进行分组。
在分组的列上我们可以使用 COUNT, SUM, AVG,等函数。
SELECT column_name, function(column_name)
FROM table_name
WHERE column_name operator value
GROUP BY column_name;
